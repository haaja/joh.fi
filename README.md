# README

My personal blog.

## Installation

Install hugo using homebrew.

```sh
brew install hugo
```

## Build the site

Generate html files

```sh
hugo -t hugo-goa
```

Test the generated site:
```hs
hugo server -t hugo-goa
```

Deploy to the site:
```sh
./deploy.sh
```
